# フロントエンド（モニター画面の描画）
言語: HTML + JavaScript + CSS

- 画面上にスマホで読み取るためのQRコードを表示しておく (スマートフォンのフロントエンドへのURL)
- パソコンのブラウザで表示するゲーム画面の描画
- バックエンドから受け取ったデータをもとに描画する
- スマホ一台につきキャラクターを追加する (定数で2人までと上限をつける、あとで変更しやすいように定数を利用する)

## バックエンドから受信するデータ
### キャラクターの座標
```json
{
  "type": "update",
  "characters": [
    {
      "id": "<端末のuuid>",
      "x": 3
    },
    {
      "id": "<端末のuuid>",
      "x": -2
    }
  ],
  "enemies": [
    {
      "id": "<敵のuuid>",
      "x": 1,
      "y": 2
    },
    {
      "id": "<敵のuuid>",
      "x": -1,
      "y": 3
    }
  ],
  "bullets": [
    {
      "id": "<弾のuuid>",
      "x": 0,
      "y": 1
    },
    {
      "id": "<弾のuuid>",
      "x": 2,
      "y": 3
    }
  ]
}
```
